# My Personal Portfolio Website

A simple and clean portfolio website built with HTML and CSS.

## 🚀 Quick Start

1. Download or clone this repository
2. Open `index.html` in your browser
3. Customize the content to make it your own

## 📝 Customization Guide

### Update Personal Information

Open `index.html` and change:

- **Name**: Find `Abdisamad Hussein` and replace with your name
- **Title**: Find `Fullstack Software & AI Engineer` and replace with your title
- **About text**: Update the paragraph in the About section
- **Projects**: Add or remove project cards in the Projects section
- **Contact info**: Update email, phone, and social links

### Change Profile Image

1. Add your photo to the `images` folder
2. Update the image path in the code:
```html
<img src="images/your-photo.jpg" alt="Profile">
